# ♻️ Descarte Consciente Brasília

Aplicação web para facilitar a localização de ecopontos e locais de descarte correto de resíduos em Brasília.

---

## 📌 Problema

A dificuldade de encontrar locais adequados para descarte de resíduos urbanos contribui para o descarte irregular e impactos ambientais.

Este projeto propõe uma solução simples e acessível para conectar usuários a pontos de coleta na cidade, incentivando o descarte consciente.

---

## 🚀 Funcionalidades

- Mapa interativo com ecopontos  
- Filtro por tipo de resíduo  
- Cálculo de distância com base na localização do usuário  
- Acesso direto à rota no Google Maps  
- Interface simples e responsiva

## 🛠️ Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript  
- Google Maps API  

---

## ⚙️ Diferenciais técnicos

- Integração com API externa (Google Maps)  
- Uso de geolocalização via navegador (Geolocation API)  
- Implementação de cálculo de distância entre coordenadas (fórmula de Haversine)  
- Manipulação dinâmica do DOM com JavaScript puro  
- Estrutura de dados em JSON para gerenciamento dos pontos de coleta  
- Ordenação de resultados com base na proximidade do usuário  

---

## 🧠 Decisões técnicas

- Aplicação 100% client-side para reduzir complexidade e custo de infraestrutura  
- Uso de JavaScript puro (sem frameworks) para maior controle e leveza  
- Estrutura de dados em JSON local para facilitar manutenção e expansão inicial  
- Cálculo de distância realizado no client para evitar dependência de serviços externos  

---

## ⚠️ Limitações atuais

- Base de dados de ecopontos limitada e estática  
- Ausência de backend para persistência e atualização em tempo real  
- Dependência de APIs externas para mapas e rotas
